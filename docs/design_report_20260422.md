# 워크북 템플릿 개발 보고서

**작성일:** 2026-04-22  
**작성자:** 유혜원  
**대상:** 레벨미업학원 워크북 자동 생성 시스템  

---

## 1. 작업 개요

기존 워크북 인쇄 템플릿을 전면 개선했습니다.  
백업 파일(1,019줄) 대비 현재 파일(2,264줄)로, 구조와 디자인이 크게 향상되었습니다.

---

## 2. 주요 변경 사항

### 레이아웃 구조 개편
- **기존:** 단순 박스 나열 방식
- **변경:** 머리글 / 본문 / 바닥글 3단 구조로 분리  
  → 페이지 번호, 솔루션 박스 위치가 일정하게 고정됨

### 인쇄 여백 최적화
- **기존:** 상하좌우 동일 여백 (상단 12mm, 하단 18mm, 좌우 15mm)
- **변경:** 상하 6mm + 홀수/짝수 페이지 제본 여백 자동 적용  
  → 양면 인쇄 시 제본 쪽에 여백이 자동으로 확보됨

### 폰트 통일
- **기존:** 폰트 크기가 페이지마다 제각각
- **변경:** 전 페이지 8pt 기준으로 통일  
  → 인쇄 시 일관된 가독성 확보

### 디자인 색상 조정
- 파란색 계열을 더 진하게 조정 → 인쇄 시 색상 선명도 향상

### 표 형식 도입
아래 두 영역을 기존 선+화살표 방식에서 **표 형식**으로 교체  
- 어휘 확인 테스트
- 오답 정리 섹션  
  → 줄바꿈 문제 해소, 시각적으로 정렬된 빈칸 제공

### Stage 5 (글의 흐름) 2단 레이아웃
- **기존:** 단순 1단 나열
- **변경:** 좌→우 순서로 채워지는 2단 컬럼 레이아웃  
  → 한 페이지에 더 많은 내용 수록 가능

### 쪽번호 방식 변경
- **기존:** `- 1 -` 형식 (인쇄 환경에 따라 표시 안 되는 문제 있음)
- **변경:** CSS 카운터 방식으로 교체 → 모든 환경에서 안정적으로 표시

### 솔루션 문구 형식 개선
- **기존:** `• 모르겠으면 제발 물어보세요.` (항목별 별도 표기)
- **변경:** `[어법 유형 솔루션] : 모르겠으면 제발 물어보세요.` (제목과 한 줄 통합)
